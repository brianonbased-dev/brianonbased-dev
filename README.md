## Building HoloScript 🌐

**[HoloScript](https://github.com/brianonbased-dev/HoloScript)** — the semantic layer for spatial computing and AI agents. Write once, run as VR, robotics, web, or game.

One semantic source compiles to **40+ targets**: Unity · Unreal · Godot · React-Three-Fiber · OpenXR / VisionOS · ROS2 / URDF · WebGPU / WASM · 3D Gaussian Splatting · glTF / USDZ · QASM, and more. Ships a 150-tool MCP server, a multi-agent memory layer, and physics + scientific solvers.

🔗 **[holoscript.net](https://www.holoscript.net/)**
